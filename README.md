# WiFi Renting Web Application

A Flask-based web application that allows users to rent WiFi access packages, pay via M-Pesa, and manage their WiFi sessions. This application integrates with Mikrotik routers to control access and supports secure user authentication and payment processing.

---

## Features

- User registration and login with secure password hashing.
- Purchase WiFi plans via M-Pesa STK Push.
- Dynamic WiFi package management per user.
- Admin dashboard for monitoring users and vouchers.
- Mikrotik router integration for hotspot management.
- Referral system with voucher validation.
- Responsive frontend using HTML, CSS, and JavaScript.

---

## Technologies Used

- **Backend:** Python, Flask, Flask-WTF, SQLite
- **Frontend:** HTML, CSS (Tailwind), JavaScript
- **Payment:** M-Pesa API integration (STK Push)
- **Router Management:** Mikrotik RouterOS API
- **Security:** CSRF protection, password hashing with Werkzeug
- **Environment Management:** Python-dotenv
